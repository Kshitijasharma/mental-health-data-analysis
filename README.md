## Mental Health Data Analysis

### Overview
This project is a web-based platform designed to provide insights into global mental health issues, with a focus on depression. It leverages interactive data visualizations and dashboards to present an in-depth analysis of mental health trends using a dataset.

### Features
- **Homepage**: Provides an introduction to depression as a common mental health issue.
- **Download Dataset**: Offers a link to download the dataset for further analysis.
- **About Section**: Details the purpose of the project and the importance of analyzing mental health data.
- **Story Section**: Interactive Tableau stories that illustrate key insights from the dataset.
- **Dashboard**: An interactive Tableau dashboard providing visual analytics on mental health trends.
- **Team Section**: Showcases the contributors to the project.

### Technologies Used
- **HTML5, CSS3, JavaScript**: Core frontend technologies.
- **Bootstrap**: Responsive design framework.
- **Tableau**: Used for interactive visualizations and dashboards.
